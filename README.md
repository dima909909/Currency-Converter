## Описание

Приложение для конвертации валют с графическим интерфейсом. Особенности:
* Получение актуальных курсов через внешний API.
* Сохранение истории конвертаций в JSON файл.
* Интуитивно понятный интерфейс.

## Как получить API ключ

1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Зарегистрируйтесь (можно использовать бесплатную учётную запись).
3. После регистрации перейдите в раздел «Dashboard» или «API Keys».
4. Создайте новый API ключ или используйте предоставленный по умолчанию.
5. Скопируйте ключ и вставьте его в код приложения:

```python
self.api_key = "ВАШ_API_КЛЮЧ_ЗДЕСЬ"
## Как запустить приложение

### Требования

* Python 3.6 или выше;
* библиотеки:
  * `tkinter` (обычно идёт в комплекте с Python);
  * `requests` (установите через `pip install requests`).

### Запуск

1. Откройте терминал или командную строку.
2. Перейдите в директорию проекта: `cd path/to/CurrencyConverter`.
3. Запустите приложение: `python main.py`.

## Примеры использования

### Тест 1: Базовая конвертация

1. Выберите валюты:
   * **Из валюты:** USD;
   * **В валюту:** EUR.
2. Введите сумму: `100`.
3. Нажмите **«Конвертировать»**.
4. Убедитесь, что в поле «Результат» отображается корректный расчёт (например, `100 USD = 92.50 EUR`).
5. Проверьте, что запись появилась в таблице истории.

## Особенности реализации

* **Использование API:** приложение запрашивает актуальные курсы через [exchangerate-api.com](https://www.exchangerate-api.com/).
* **Сохранение данных:** история конвертаций сохраняется в `conversion_history.json` между запусками.
* **Валидация:** проверка корректности суммы и выбора валют.
* **Интуитивный интерфейс:** все элементы управления расположены логично и понятно.
